== Description ==

The Push.co plugin integrates your Wordpress blog with your Push.co app. Send Push notifications for all new posts, straight from your blog.

== Installation ==

1. Create an app on Push.co
2. Upload the push.co folder to the /wp-content/plugins/ directory
3. Activate the Push.co plugin through the 'Plugins' menu in WordPress
4. Configure the plugin by going to the plugin settings

== Frequently Asked Questions ==

= What is Push.co =

Push.co is a simple way to receive 3rd party push notifications in one app. Push.co allows you to subscribe to 3rd party push apps and allows developers to send push notifications to their fans and/or users in a simple way without the need of having a mobile app themselves.

= Where can I read more about Push.co =

You can read more about Push.co on https://push.co.
